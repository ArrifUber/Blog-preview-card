### What I learned

I learned how to use hover transition in CSS

```css
.card:hover {
  box-shadow: 16px 16px rgb(0, 0, 0);
  .title {
    color: var(--color-yellow);
    cursor: pointer;
    transition: all ease 600ms;
  }
}
```
